HOW TO START
LEARNING
CODING

Learning coding takes time, it demands patience and motivation to never give up in spite of challenges on the way. If you are able to persist to understand the concept it will later become a fun thing for you to do.
The first and foremost step you must take is to choose the right language to learn. However, before you do that you may need to learn markup structure like HTML. It is an important language for any programmer to learn and have a glance of the concept of HTML whether you want to become a web developer, apps developer, a game developer, or simply writing useful programs. HTML stands for "Hypertext Markup Language". It is the language you will need to build structure for your programs, this is because HTML is the skeleton for website apps and other computer programs.

BEAUTY OF WEBSITE

The next thing you may need to have a glance at is the styling sheet, that is CSS. CSS stands for "Cascading Style Sheet". it is what you will use to beautify your website or apps and make them look great to use. After you have gotten some knowledge of HTML and CSS and their framework, the next thing is to choose the right programming language for yourself. For example if you want to specialize in web development the best way to go is to learn JavaScript. If on the other hand you want to develop web apps you can start with python which is easier to learn.
